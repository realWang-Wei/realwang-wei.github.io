二叉树

二叉树是一种数据结构，有时也会考。比如有的面经里就提到了考中序遍历，二叉树的遍历都是很基础的，如果这都写不出来基本凉凉。

  • 前序遍历：根 -> 左 -> 右
  • 中序遍历：左 -> 根 -> 右
  • 后序遍历：左 -> 右 -> 根
  • 层次遍历：BFS
